Methodology

This document describes assumptions, expected outcomes, and known limitations for our test patterns.

Dead Pixel and Uniformity

Solid color and low-gradient patterns reveal non-functional subpixels and edge bleed. Ambient light and panel warm-up state can influence perceived severity.

Motion and Cadence

Browser frame pacing diagnostics estimate practical smoothness by sampling animation frame deltas. These measurements are not a substitute for hardware oscilloscope latency tests.

Input Lag Trainer

Reaction timing includes human variance, keyboard/mouse latency, browser scheduling, and display response. Results are useful for comparative tuning, not absolute esports certification.

HDR Clipping Simulator

This mode uses SDR-safe highlight ramps to expose clipping tendencies in tone-mapping chains. It helps detect flattened highlight detail but cannot report exact peak luminance in nits.

Subpixel Inspector

Magnified RGB patterns are intended for visual inspection of subpixel layout and text fringe behavior. The on-screen result varies with browser scaling and OS text smoothing settings.
